## 一键安装 稳定版 使用说明

### 温馨提示: 连续两次 Ctrl + C 可快捷终止程序运行

### 声明：
- 此脚本完全绿色，可以放心使用。
        

### 普通用法

- 1、
    将install_zh_cn.exe文件放入到原wiz补丁安装目录下
- 2、
    创建一个快捷方式，快捷方式可随意命名，放到桌面或自己方便寻找的位置
- 3、
    双击快捷方式运行
- 4、
    根据程序指引来输入对应类型补丁进行安装

#### 重置功能介绍：将所有补丁全部删除，重置为原版

两种启动方式

    - cmd启动  -h 可查看所有支持的参数(一般用不到)

    - 双击启动  可自由输入对应的补丁类型进行安装和更新
